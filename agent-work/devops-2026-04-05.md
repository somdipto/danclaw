## Minor Observations

### Inconsistency Found
- `apps/web/.env.example` uses `NEXT_PUBLIC_OPENROUTER_KEY`
- `apps/web/.env.local.example` uses `NEXT_PUBLIC_OPENROUTER_KEY` 
- `apps/mobile/.env.example` uses `EXPO_PUBLIC_OPENROUTER_API_KEY`
- `docs/DEPLOY.md` references `NEXT_PUBLIC_OPENROUTER_API_KEY`

Should standardize on `NEXT_PUBLIC_OPENROUTER_API_KEY` vs `NEXT_PUBLIC_OPENROUTER_KEY` across all web files.

### ci.yml Build Dependencies
The `build-web` job depends on `[lint, typecheck]` but lint/typecheck don't actually filter by package. This means they run full workspace commands even though only web is being built. Minor inefficiency but not breaking.
